# RudiSmartShop

Proyecto personal de e-commerce construido con Next.js, Tailwind CSS, MongoDB y autenticación JWT.

## Funcionalidades

1. Gestión CRUD para admins, usuarios, productos y órdenes
2. Tema claro/oscuro
3. Soporte bilingüe (EN/FA)
4. Panel de administración con niveles de acceso
5. Filtros y ordenamiento en búsqueda

## Desarrollo local

```bash
npm install
npm run dev
```
